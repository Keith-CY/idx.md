  # Command Reference

  Complete reference for all agentic-browser functions. For quick start, see
  [SKILL.md](../SKILL.md).

  ## Base Command

  All commands follow this pattern:

  ```bash

  infsh app run agentic-browser --function <function> --session <session_id|new>
  --input '<json>'

  ```

  - `--function`: Function to call (open, snapshot, interact, screenshot,
  execute, close)

  - `--session`: Session ID from previous call, or `new` to start fresh

  - `--input`: JSON input for the function
